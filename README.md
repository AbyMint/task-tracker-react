# task-tracker-react

Simple React task tracker with local storage persistence.

## Features
- Add, edit, and delete tasks
- Mark tasks complete/incomplete
- Filter tasks by all/completed/pending
- Optional subtasks per task
- Task completion is blocked until all subtasks are completed (when subtasks exist)
- Notes field on each task
- Now with dark mode!

## Run
```bash
npm install
npm run dev
```

## Build and lint
```bash
npm run lint
npm run build
```

## Tests
```bash
npm run test
```

Run all quality checks (lint, tests, production build):
```bash
npm run check
```
